# Terminal Notes
Use the terminal to create simple notes


## Setup
1. chmod +x setup.sh
2. ./setup.sh
3. restart terminal


## Alias
* an - add a note
* sn - show notes
* dn - delete a note


## Usage

add the note "this is my first note"
> an this is my first note

show all notes
> sn

delete note number 3
> dn 3
